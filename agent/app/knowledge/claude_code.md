# Claude Code 使用技巧

## 基本启动

- 在项目根目录运行 `claude` 即可启动交互式会话
- 使用 `claude -p "你的指令"` 可以非交互模式直接执行任务
- 使用 `claude --resume` 恢复上次会话

## 高效交互技巧

1. **明确意图**：告诉 Claude 你要做什么，而不是怎么做。例如："重构 auth 模块，把 JWT 验证提取为独立中间件"
2. **分步执行**：复杂任务拆分为多个步骤，每步确认后再继续
3. **提供上下文**：用 `@文件名` 引用特定文件，帮助 Claude 理解你的代码
4. **使用斜杠命令**：
   - `/help` 查看所有命令
   - `/compact` 压缩对话上下文
   - `/clear` 清空当前对话
   - `/cost` 查看 token 使用量
   - `/review` 让 Claude 审查代码变更

## CLAUDE.md 配置

在项目根目录创建 `CLAUDE.md` 文件，可以为 Claude 提供项目特定的上下文和规则：

```markdown
# 项目说明
## 技术栈
- 后端: Python FastAPI
- 前端: Flutter

## 编码规范
- 使用中文注释
- 函数必须有类型注解
```

## 常见场景

- **代码重构**：描述重构目标，Claude 会分析依赖关系并提出方案
- **Bug 修复**：描述现象和复现步骤，Claude 会定位问题
- **写测试**：指定要测试的模块和覆盖要求
- **代码审查**：使用 `/review` 命令或让 Claude 检查特定文件
