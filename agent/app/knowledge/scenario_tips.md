# 场景化编程建议

## 代码重构

启动 Claude Code 时建议附带以下提示：

```
请帮我重构 [模块名]，目标是 [重构目标]。
要求：
- 保持现有测试通过
- 每次只做一个小改动
- 改动后立即运行测试验证
```

## 写测试

启动 Claude Code 时建议附带以下提示：

```
请为 [模块名] 编写测试，覆盖以下场景：
- 正常路径
- 边界条件
- 异常处理
使用 [测试框架]，测试文件放在 [路径]
```

## 排查 Bug

启动 Claude Code 时建议附带以下提示：

```
遇到以下问题：
- 现象：[描述]
- 复现步骤：[步骤]
- 期望行为：[期望]
请帮我定位根因并修复
```

## 新功能开发

启动 Claude Code 时建议附带以下提示：

```
请实现 [功能描述]，要求：
- 遵循项目现有架构模式
- 先写测试再实现
- 注意边界条件和错误处理
```
