## 脚本功能

### 命令行参数

- `./build-desktop.sh` — 完整构建
- `--skip-agent` — 跳过 Agent 构建，使用已有 agent/dist/rc-agent
- `--skip-flutter` — 跳过 Flutter 构建，使用已有 .app
- `--dmg-only` — 仅从已有产物生成 DMG
- `-h/--help` — 显示帮助

### 构建流程

1. **Step 0**: 前置检查 — pyinstaller, flutter, hdiutil, codesign
2. **Step 1**: cd agent && pip install -r requirements.txt && pyinstaller rc-agent.spec
3. **Step 2**: cd client && flutter pub get && flutter build macos --release
4. **Step 3**: 创建 .app/Contents/Resources/agent/ 目录，复制 agent/dist/rc-agent
5. **Step 4**: chmod +x rc-agent
6. **Step 5**: codesign --force --deep --sign - rc_client.app
7. **Step 6**: hdiutil create 生成 DMG（含 Applications symlink）
8. **Step 7**: 三项自动化验证

### 自动化验证

1. .app bundle 内 rc-agent 存在且有执行权限
2. codesign --verify --deep --strict 通过
3. hdiutil attach 后验证：.app 存在、rc-agent 可执行、Applications symlink 存在
