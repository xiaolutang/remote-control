## 根因分析

> fix_level: L2

**现象**: 服务器重启后 Agent 收到 1012 (service restart)，进程永久挂起不退出也不重连。

**根因链**:
```
websocket_client.py:356 — asyncio.gather(*self._tasks) 阻塞
  ├── _websocket_to_pty: 检测到 1012 → break（不设 _connected=False）✗
  ├── _heartbeat_loop: 检测到 1012 → break（不设 _connected=False）✗
  └── _pty_to_websocket: while self._running and self._connected → 永远循环 ✗
gather 死锁 → _connect_and_run() 不返回 → run() while 循环不进入重试
```
