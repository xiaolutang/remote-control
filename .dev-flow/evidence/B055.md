### 1. 新增 5 个效率指标

| 指标名 | 计算方式 | 数据来源 |
|--------|----------|----------|
| n_turns | phase_change 事件计数 | phase_change_events |
| n_toolcalls | tool_step 事件计数 | tool_step_events |
| time_to_first_token | phase_change(thinking) → 首个 streaming_text 时间差 | thinking_phase_time + first_streaming_text_time |
| output_tokens_per_sec | output_tokens / streaming_duration | output_tokens + first/last_streaming_text_time |
| time_to_last_token | 首事件 → result/error 时间差 | first_event_time + result_or_error_time |

### 4. 关键设计决策

- **无法计算的时间指标返回 -1.0**（而非 0.0），以便区分"无法计算"和"瞬间完成"
- **integration 事件由 harness 在循环中实时采集**，而非事后从 transcript 转换（确保时间戳准确）
- **source 过滤在 DB 层实现**（_build_metric_conditions），query 和 aggregate 共享
