## 设计决策

- **Mixin 模式**: Database 类继承 6 个 Mixin，保持单一类接口不变
- **DDL 抽取到 _schema.py**: init_db 中的 CREATE TABLE 语句占 250+ 行，提取为数据列表
- **conversation_store.py 使用 deferred import**: `AgentConversationConflict` 在方法内 import，避免循环依赖
