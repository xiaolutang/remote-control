# F126 Evidence: client 目录分层 — 桌面端/移动端代码分离

## 任务摘要

将 9 个 desktop 专属 service 文件移入 `services/desktop/` 子目录，将桌面专属 screen 移入 `screens/desktop/`，用 `TerminalViewConfig` 替代 `terminal_screen.dart` 中 12 处散落的 `isMobilePlatform` if-else。

## 变更清单

### 1. services/desktop/ 目录分层

**移动文件（git mv 保留历史）：**
- `desktop_agent_bootstrap_service.dart`
- `desktop_agent_exit_bridge.dart`

- `services/desktop/desktop_services.dart` — 统一导出所有 desktop service

### 2. screens/desktop/ 目录分层

**移动文件（git mv 保留历史）：**
- `terminal_workspace_screen.dart` → `screens/desktop/`

**未移动（shared）：**
- `terminal_screen.dart` — 保留在 `screens/`（桌面端和移动端共用）
- `terminal_screen_controller.dart` — 保留在 `screens/`（共用逻辑）

### 3. PlatformDelegate — TerminalViewConfig

**新增文件：**
- `services/terminal_view_config.dart` — `TerminalViewConfig` 类，集中管理所有平台相关的 UI 配置

**terminal_screen.dart 变更：**
- 12 处 `_ctrl.isMobilePlatform` if-else 全部替换为 `_viewConfig` 属性读取
- 删除 `_mobileTerminalStyle()` 方法（迁移到 `TerminalViewConfig._mobileStyle()`）
- 构造阶段创建 `TerminalViewConfig.forPlatform(platform)` 实例

**集中管理的决策点（原 12 处 → 0 处散落）：**
| 配置项 | 原代码 | 新代码 |
|--------|--------|--------|
| 字体样式 | `_ctrl.isMobilePlatform ? _mobileTerminalStyle(p) : const TerminalStyle(...)` | `vc.textStyle` |
| 自动聚焦 | `!_ctrl.isMobilePlatform` | `vc.autofocus` |
| 输入行为 | `_ctrl.isMobilePlatform ? send : newline` | `vc.inputAction` |
| 建议开关 | `_ctrl.isMobilePlatform` | `vc.enableSuggestions` |
| IME 个性化 | `_ctrl.isMobilePlatform` | `vc.enableIMEPersonalizedLearning` |
| TUI 选择器 | `if (_ctrl.isMobilePlatform)` | `if (vc.showTuiSelector)` |
| 快捷键栏 | `if (_ctrl.isMobilePlatform && ...)` | `if (vc.showShortcutBar && ...)` |
| resizeToAvoidBottomInset | `!_ctrl.isMobilePlatform` | `vc.resizeToAvoidBottomInset` |
| 重连后聚焦 | `!_ctrl.isMobilePlatform` | `!vc.requestFocusAfterRetry` |
