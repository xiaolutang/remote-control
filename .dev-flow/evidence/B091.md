## Acceptance Criteria

1. Agent 收到 lookup_knowledge WebSocket 消息后能正确检索并返回知识内容
2. 内置知识文件包含：Claude Code 使用技巧、Vibe Coding 方法论、4 类场景建议
3. 支持用户自定义知识文件目录
4. 关键词匹配能命中相关文件
5. 未匹配时返回'未找到相关知识'而非报错
6. 检索结果裁剪：最多 3 个文件，每文件最多 2000 字符，截断标注
7. knowledge/ 随 Agent 打包；user_knowledge/ 缺失时自动创建
8. lookup_knowledge 消息在 websocket_client.py 现有框架内扩展
9. knowledge_config.json 配置存储，默认全启用，缺失/损坏时全启用
10. 配置变更下次启动生效，R043 不做热更新
11. Agent 启动时上报 tool_catalog_snapshot 含 lookup_knowledge built-in entry
12. 桌面本地 Agent 路径：knowledge/ 随分发包
