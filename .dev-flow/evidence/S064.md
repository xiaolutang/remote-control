## 发现的问题

### Agent 1012 重连阻塞（非 S064/S065 范围）

**现象**: 服务器重启后 Agent 收到 1012 (service restart)，但不会自动重连。
**根因**: `_connect_and_run()` 内部 `asyncio.gather()` 等待所有任务完成，但 1012 断连时只有 `_websocket_to_pty` 和 `_heartbeat_loop` 退出，`_pty_to_websocket` 仍等待 PTY 输出，阻塞了 gather → `_connect_and_run()` 无法返回 → `run()` while 循环无法进入下一次重连。
**影响**: Agent 需要依赖 Docker `restart: unless-stopped` 策略恢复（进程退出才重启），但 1012 不会触发进程退出。
**建议**: 后续补充修复 — 任务断连后应设置 `_connected = False` 通知 PTY 任务退出，或使用 `asyncio.wait(return_when=FIRST_EXCEPTION)` 替代 gather
