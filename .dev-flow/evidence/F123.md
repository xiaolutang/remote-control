## 提取清单

### 1. StatusPill → `lib/widgets/shared/status_pill.dart`

**重复来源**（3 处）：
- `lib/screens/runtime_selection_screen.dart` — icon+label 模式，独立 `_StatusPill` 类
- `lib/widgets/smart_terminal_create_dialog_preview.dart` — label-only 模式，带 backgroundColor/textColor 的 `_StatusPill`
- 同上文件 — `_ExecutionStatusPill` 对 `_StatusPill` 的二次包装

**提取方案**：统一为 `StatusPill` widget，支持两种模式：
- icon 模式：传 `icon` + `label`
- label-only 模式：传 `label` + `backgroundColor` + `textColor`

**修改文件**：
- `lib/screens/runtime_selection_screen.dart` — 删除 `_StatusPill` 类，改用 `StatusPill`
- `lib/widgets/smart_terminal_create_dialog_preview.dart` — 删除 `_StatusPill` 类，改用 `StatusPill`

### 2. ChatBubble → `lib/widgets/shared/chat_bubble.dart`
