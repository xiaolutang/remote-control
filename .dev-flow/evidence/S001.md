# S001 Evidence: AgentResult 扩展调度字段

## 任务概要

在 AgentResult 模型中增加可选的 `schedule_at`（带时区 ISO 8601）和 `repeat_type`（once|daily）字段，支持定时命令。

### Findings

1. **[correctness/low]** schedule_at 时区校验使用正则 `+xx:xx` 或 `Z` 后缀，足够覆盖 ISO 8601 常见格式。未验证日期时间数值合法性，这是有意设计：时间解析交给消费端。
2. **[completeness/low]** repeat_type 使用 `Literal["once", "daily"]` 由 Pydantic 强约束，非法值直接 ValidationError。
3. **[backward_compatibility/low]** 新增字段均为 Optional，默认 None。SSE result_event_data 使用条件透传（仅在非 None 时加入 dict），完全向后兼容。
4. **[test_coverage/low]** 13 个新测试覆盖全部 10 个 test_tasks。
