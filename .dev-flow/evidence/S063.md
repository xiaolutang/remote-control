### 禁止模式 #105 守卫

- agent/websocket_client.py:538-539: `logger.error("Encrypt failed, dropping message (not sending plaintext)")` — 不降级为明文
- client/auth_service.dart: ws:// + 公钥失败 → 直接 throw，不回退 `body['password'] = password`
- agent/auth_service.py: ws:// + 公钥失败 → `raise`，异常传播给调用方

### wss:// 路径（不变量 #29）

- wss:// 连接由 TLS 保护，不做应用层加密
- 登录/注册：公钥获取失败可回退到明文密码（TLS 已加密传输）
- WS：AES 密钥交换为可选，失败后消息走明文（TLS 保护）
