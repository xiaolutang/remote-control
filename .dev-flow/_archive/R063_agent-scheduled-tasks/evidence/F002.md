# F002: Agent 面板定时确认 UI — Evidence

## 实现概要

当 Agent 返回带 `scheduleAt` 的 command result 时，展示定时确认卡片（而非立即执行按钮）。

## 变更文件

### 新增文件
- `client/lib/models/scheduled_task.dart` — `ScheduledTaskRepeatType` 枚举（once/daily），`fromString()` 降级逻辑
- `client/lib/services/scheduled_task_service.dart` — `ScheduledTaskService`，对接 `POST /api/scheduled-tasks`
- `client/test/mocks/mock_http_client.dart` — MockHttpClient 测试工具
- `client/test/models/scheduled_task_test.dart` — 枚举单元测试（9 cases）
- `client/test/services/scheduled_task_service_test.dart` — Service 单元测试（6 cases）
- `client/test/widgets/agent_panel_schedule_test.dart` — Widget 测试（8 cases）

### 修改文件
- `client/lib/widgets/agent_panel_result_views.dart` — 新增 `_buildScheduleConfirmCard`、`_formatScheduleTime`、`_formatRepeatType`；`_buildCommandResultView` 在 `scheduleAt != null` 时路由到定时确认卡片
- `client/lib/widgets/agent_panel_handlers.dart` — 新增 `_createScheduledTask`：steps 用 `\r` 拼接 text_content，repeatType 转枚举，调 ScheduledTaskService.create()，成功后 SnackBar + onScheduledTaskCreated 回调
- `client/lib/widgets/agent_panel_state.dart` — 新增 `_scheduledTaskCreating`、`_scheduledTaskError` 状态字段及 `_createScheduledTask` stub
- `client/lib/widgets/smart_terminal_side_panel.dart` — SmartTerminalSidePanel 新增 `onScheduledTaskCreated` 可选回调
- `client/lib/widgets/smart_terminal_side_panel_content.dart` — _SmartTerminalSidePanelContent 新增 `onScheduledTaskCreated`，声明状态字段
- `client/lib/screens/terminal_screen.dart` — TerminalScreen 新增 `onScheduledTaskCreated` 参数，传递给 SmartTerminalSidePanel
- `client/lib/screens/desktop/terminal_workspace_screen.dart` — _buildTerminalView 传递 onScheduledTaskCreated（暂 null，等 poller 实现）
