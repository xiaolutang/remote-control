## 关键设计决策

### session_id 命名映射
- `AgentDeps.session_id` = WS session id（不用于 store 查询）
- 闭包中 `device_id` 映射为 `ScheduledTaskStore` 的 `session_id` 字段
- 这是因为 store 表设计中 `session_id` 语义为 device 级标识

### 取消操作的三重归属校验
- `task.user_id == 当前 user_id`
- `task.session_id == 当前 device_id`
- `task.terminal_id == 当前 terminal_id`
- 任一不匹配即返回越权错误，不执行删除

### System Prompt 时间注入
- `_build_system_prompt()` 每次调用获取 UTC+8 当前时间
- 要求 Agent 输出绝对 ISO 8601 时间（如 `2026-05-14T03:00:00+08:00`）
- 禁止使用相对时间
