## 实现细节

### 字段新增
- `AgentResultEvent.scheduleAt` (String?) — 调度执行时间，ISO 8601 格式
- `AgentResultEvent.repeatType` (String?) — 重复类型 (daily/hourly/weekly 等)

### JSON 映射
- `fromJson`: 使用 `readOptionalStringFromJson` 解析 `schedule_at` / `repeat_type`
- `toJson`: 仅在非 null 时序列化 `schedule_at` / `repeat_type`
- 非法值处理: 非字符串类型返回 null，字符串类型保留原值（含非 ISO 8601）

### 遵循的约束
- JSON 解析通过 `json_helpers.dart` 的 `readOptionalStringFromJson`
- models/ 不依赖 Flutter UI 框架
- 字段可选，不影响现有解析链路
