# F050: Fix terminal injection truncation with Bracketed Paste Mode

## 变更内容

**文件**: `client/lib/services/ws_connection_manager.dart`

**方法**: `_wsSend` (line 169)

**修改**: 在 base64 编码前，检测 `data` 是否包含换行符 `\n`。如果包含，用 Bracketed Paste Mode 转义序列包裹内容：

- 前缀: `\x1b[200~`
- 后缀: `\x1b[201~`
- 单行内容（不含 `\n`）不加包裹

## 修改逻辑

```dart
final content = data.contains('\n')
    ? '\x1b[200~$data\x1b[201~'
    : data;
```
