### Key Implementation Decisions

1. **StatefulWidget + MouseRegion**: 使用 MouseRegion.onEnter/onExit 检测鼠标悬停状态，AnimatedContainer 动画过渡宽度（48px <-> 160px, 200ms easeInOut）。
2. **接口与 TerminalTabBar 一致**: terminals, selectedTerminalId, onSwitch, onCreate, createDisabled, onContextMenu 签名完全一致，便于后续 F003 替换。
3. **TerminalItem 使用 canAttach 控制可切换性**: 与 TerminalTabBar 行为一致，closed terminal 渲染但不可切换，图标 dimmed。
4. **折叠态 Row 布局**: indicator(3px) + SizedBox(icon, 44px) = 47px，适配 48px AnimatedContainer 宽度减去 1px 右边框。
5. **右键使用 onSecondaryTapDown**: 与 TerminalTabBar 保持一致，提供 globalPosition 供 PopupMenu 定位。
6. **需要 import gestures.dart**: PointerEnterEvent/PointerExitEvent 在 material.dart 中未导出，需显式导入 `package:flutter/gestures.dart`。
