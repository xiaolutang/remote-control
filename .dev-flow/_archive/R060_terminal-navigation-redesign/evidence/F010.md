## Acceptance Criteria
1. IndexedStack 为每个终端缓存独立的 TerminalScreen
2. 每个终端持有独立的 WebSocketService 实例（通过 TerminalSessionManager + per-child Provider）
3. 刷新终端列表时，selectedIndex 按 terminalId 匹配而非位置索引
4. 终端顺序变化后，selectedIndex 正确跟随 terminalId
5. 关闭选中终端时，IndexedStack 切换到下一个而非 clamp 到 0

## Implementation

### 测试覆盖
7 个 widget tests 覆盖以下场景：
1. `terminal!=null → IndexedStack visible (no CircularProgressIndicator)`
2. `loadingTerminals=true + terminal!=null → IndexedStack visible`
3. `2 terminals each hold independent WebSocketService instances`
4. `switch terminal → IndexedStack index changes but children count and keys stay same`
5. `(replace terminals list) → selectedIndex still points to original terminalId`
6. `3 terminals + reorder → selectedIndex follows terminalId not position`
7. `close selected terminal via context menu → IndexedStack switches to next, not clamped to 0`
