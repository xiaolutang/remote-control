## Acceptance Criteria

1. 新增 `_pendingReset` 布尔状态字段
2. conversation_reset SSE 活跃时设置 `_pendingReset = true` 而非静默丢弃
3. SSE onDone 时检查 `_pendingReset`：为 true 时清空 `_serverConversationEvents / _agentHistory / _activeSessionId` 并重置 `_nextConversationEventIndex`
4. pendingReset 处理后标记 `_pendingReset = false`
5. 测试覆盖：SSE 活跃 → conversation_reset → SSE 结束 → 状态正确重建
6. 测试覆盖：SSE 活跃 → 无 conversation_reset → SSE 结束 → 状态不变
7. 测试覆盖：pendingReset 清空后 conversation stream 增量同步正确重建 UI
