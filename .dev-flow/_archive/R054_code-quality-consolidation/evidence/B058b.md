## 架构决策

user_api.py 使用 `from app.infra import auth as _auth` 和 `from app.store import refresh_token_store` 的模块引用方式调用函数，而非 `from X import Y`。这使得测试可以直接 patch 源模块路径（如 `app.infra.auth.verify_password`），避免需要同时 patch 源模块和消费模块。
