# B056 Evidence: agent_request 统一错误处理 + pending registry

## 变更摘要

将 agent_request.py 中 6 个 pending dict 封装为 PendingRequestRegistry 类，统一管理 future 生命周期和超时清理。错误处理策略分为两类：
- 关键路径（create_terminal、close_terminal_with_ack、execute_command）统一抛 HTTPException
- best-effort 路径（snapshot、lookup_knowledge、tool_call）返回空/error dict 并记录 warning
