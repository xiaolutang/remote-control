# B058: ws_client 拆分

## 变更摘要

将 `server/app/ws/ws_client.py`（632 行）拆分为 4 个子模块 + 1 个入口模块，与 ws_agent 拆分模式一致。

## 拆分结构

| 文件 | 行数 | 职责 |
|------|------|------|
| `ws_client.py` | 339 | 入口模块：handler 注册、生命周期管理、连接建立/清理 |
| `client_connection.py` | 92 | ClientConnection 类、active_clients 字典、注册/注销工具函数 |
| `client_message_handler.py` | 104 | DATA/RESIZE 消息处理逻辑 |
| `client_snapshot.py` | 76 | 终端快照请求与发送 |
| `client_presence.py` | 94 | 视图计数、广播、presence 更新 |

## 设计决策

1. **不做 re-export**（B057 模块边界政策）：ws_client.py 的 import 仅供自身 handler 使用
2. **子模块直接从 store 层 import**：client_snapshot.py 直接导入 `get_session_terminal`、`get_terminal_output_history` 等
3. **handler 入口保留在 ws_client.py**：`client_websocket_handler` 和 `_cleanup_client` 保留在入口模块
