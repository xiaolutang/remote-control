## SYSTEM_PROMPT 关键变更

1. **移除 ask_user 强制确认流程**：不再主动追问 "Claude Code 在运行吗？"
2. **简化 response_type 选择规则**：
   - command：用户要求执行终端操作（启动工具、终端命令、用 Claude Code 做 X）
   - ai_prompt：仅用于用户明确要求"注入/发送 prompt"
   - message：纯信息型回复
3. **明确 execute_command vs deliver_result 边界**：
   - 用户要求的操作 → deliver_result(response_type='command')
   - 自己探索环境 → execute_command
4. **收窄 ai_prompt 使用场景**：仅用户明确要求时使用
