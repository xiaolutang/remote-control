# S503: Agent sys.exit(1) 改为异常 + 清理

## 变更概述

将 `websocket_client.py run()` 方法中重连耗尽后的 `sys.exit(1)` 调用改为抛出 `ReconnectExhausted` 异常，由 `cli.py` 顶层捕获后执行清理再退出。

## 修改文件

| 文件 | 变更 |
|------|------|
| `agent/app/transport/websocket_client.py` | 新增 `ReconnectExhausted` 异常类；`run()` 中 `sys.exit(1)` 改为 `raise ReconnectExhausted` |
| `agent/app/cli.py` | 导入 `ReconnectExhausted`；`_run_with_signal_handling` 中捕获该异常，执行 `client.stop()` 清理后 `sys.exit(1)` |
| `agent/tests/test_defect_escape_config.py` | 更新 4 个测试：从 mock `sys.exit` 改为 `pytest.raises(ReconnectExhausted)` |

## 验收标准

- [x] 新增 `ReconnectExhausted` 异常类（含 retry_count、max_retries 上下文信息）
